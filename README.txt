BandWagon — 실행 방법
=====================

1) 이 폴더(BandWagon) 안에 run.pyw, run.py, 그리고 bandwagon 폴더가
   함께 들어 있어야 합니다. 폴더째로 옮기세요(파일만 따로 빼지 마세요).

2) 처음 한 번, 필요한 패키지를 설치합니다 (명령 프롬프트/터미널):
       pip install PyQt5 Pillow numpy scipy opencv-python

3) 실행:
   - 더블클릭:        run.pyw   (콘솔 창 없이 GUI만)
   - 명령으로 실행:    python run.py          (콘솔 함께 — 오류 확인용)
                      python run.py 이미지.png   (이미지를 열면서 시작)

문제가 생기면 run.pyw가 작은 오류 창을 띄워 원인을 알려줍니다.
그래도 안 되면 "python run.py" 출력 전체를 복사해 알려주세요.
